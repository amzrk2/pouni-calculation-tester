# 四则运算自测系统

项目由 [Create React App](https://github.com/facebook/create-react-app) 创建。

## 内容

设计一款四则运算的自测软件，可用于小学生的四则运算训练。

在线预览：https://amzrk2.cc

### 基本思路

设置页面根据设置进行题目生成后，将结果于题目存于 store 中。做题完成后进行比对，结果保存至 store 同时于 localStorage 进行同步，提供结果下载。
